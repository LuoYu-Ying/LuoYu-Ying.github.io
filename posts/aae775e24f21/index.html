Gallery 相册图库

图库页面只是普通的页面，你只需要：

  1. hexo n page xxxxx 创建你的页面。

  2. 在 index.md 使用标签外挂 galleryGroup。

galleryGroup 模板

参数说明：

  • name：图库名字

  • description：图库描述

  • link：连接到对应相册的地址

  • img-url：图库封面的地址

示例

Gallery 相册子页面

子页面也是普通的页面，你只需要：

  1. hexo n page xxxxx 创建你的页面。
  2. 然后使用标签外挂 gallery。

gallery 模板

参数说明：

  • markdown 图片格式： ![]() 或者 <img src = ""> 的格式皆可。

示例

提示

如果你想要使用 /gallery/wallpaper 这样的链接显示你的图片内容。你可以把创建好的 wallpaper 整个文件夹移到 gallery 文件夹里中。
