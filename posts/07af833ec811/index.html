无版权图库

名称 简介
Pexels 每一张都高清大图，无版权可任性下载
Stock Up 一个收集了来自 27 个免版权图库的神奇网站，图片数量之大，超乎你的想象
Unsplash 主要是生活中的景象作品
Pixabay 支持中文搜索的免费可商用图库
StockSnap 最先进的检索功能，瞬间拥有高质量的免费高清图片
Magdeleine 主要类别有自然、城市建筑、动物、视频风格偏厚重，比较艺术化
Visual Hunt 通过颜色找图片，更精准的满足用户需求

无版权字体

地址 描述
Free Fonts 字体 emm。忘了介绍是啥了，自己打开看看吧
Google Fonts 谷歌字体中文版 谷歌字体（暂时无法使用中文字体，请访问旧版）
Adobe Fonts Adobe 的免费开源字体
站酷字库 - 付费字体 & 免费字体 - 站酷 (ZCOOL) 站酷字体
字体天下 - 提供各类字体的免费下载和在线预览服务 这个里面的免费字体好像只能用于设计，比如 PS 什么的

文章封面设计

地址 描述
文章封面生成工具 GitHub.io 的，某些地区可能无法访问
Stories 免费插图设计 自定义更改插图颜色、插图背景、隐藏元素等，插图动画化，完全免费，但必须注明来源。
Vector Creator 插图设计 有人物、动物、植物等类别的元素，而且每种类型的插图还有不同的风格，无需注册即可免费设计并下载 PNG 格式的插图，不付费使用时需要遵循免费使用条款
unDraw - 插图设计 风格单一，但完全免费，不用担心版权，可任意使用不用担心律师函
ManyPixels 插图设计 可以说是另一个版本的 unDraw 插图库，完全免费，不用担心版权

ACG 无版权图库

地址 简述
Alpha Coders ACG 无版权图片 偏向游戏、人物
GameWallpapers. ACG 无版权图库 同上

代码托管

名称 描述
Git 分布式版本管理工具，程序员必会技能
码云 Gitee 国内版的 Github，支持静态站点配置，不过付费才嫩自动部署，自定义域名
Coding 也是国内的，支持静态站点、自定义域名、强制 https、自动部署，非常好用
GitHub 国外的，功能非常全，程序员最爱的 “交友” 网站，唯一的缺点是在国外，访问有点慢
